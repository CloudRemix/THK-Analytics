PHP埋め込み用コード

  • 解析用タグ（Javascript）との併用はできません。
  • Javascript では解析できないタイプのクローラー（検索ロボット）等のアクセスも漏れなく収集できます。
  • Javascript が OFF のブラウザからのアクセスでもリファラー（リンク元）を取得できます
  • PHP埋め込み用コードでは、以下の内容については解析できません。
    • リンククリック
    • ボタンクリック
    • Adsense クリック関連項目
    • 画面解像度
    • 画面色数
    • Javascript / Cookie
    • 各ページのタイトル（ページタイトルを取得したい場合は、後述のタイトル設定用コードをご利用ください）
  • PHP埋め込み用コードでは、外部サーバー（ とは別のサーバー）にあるサイトを解析することはできません。
  • http で始まる include 文はご利用頂けません。

PHP埋め込み用コード（タイトル設定用）

  • _thkTrackのパラメータにタイトルを渡すことで、各ページのタイトルを解析結果に表示できます。
    • （例）_thkTrack('トップページ');
  • WordPress 等の場合、変数や関数を設定することで各ページのタイトルを解析結果に表示できます。
    • （例1）_thkTrack( wp_title('-', false, 'left') ); ← WordPress の場合。
    • （例2）_thkTrack( $title ); ← PHP で作成されたページなどの場合。
